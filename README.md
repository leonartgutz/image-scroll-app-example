# Simple image scroll app

A simple REST Api for a todo list with GraphQL. The GraphQL is just another way to use this app, I am still new to GraphQL but this is a good example for show my knowledge

## Install Expo Cli

```sh
$ yarn add -g expo-cli
```

## Installation

```sh
$ yarn install
```

## Scripts

```sh
$ yarn ios
```
